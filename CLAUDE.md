# Git 提交与署名规则

1. Claude Code 或任何 AI 助手在生成、修改、补全 git commit message 时，禁止添加任何 AI 署名、共同作者信息、会话链接或生成来源说明。

2. commit message 中禁止出现以下内容：
   - `Co-Authored-By: Claude ...`
   - `Co-authored-by: Claude ...`
   - `noreply@anthropic.com`
   - `Claude-Session: ...`
   - `Generated with Claude Code`
   - `🤖 Generated with ...`

3. 即使代码、文档或提交内容由 Claude Code 协助生成，提交作者也只能使用用户本人的 git identity，除非用户明确要求添加共同作者。

4. 执行 `git commit` 前，必须检查最终 commit message，确认没有任何 Claude、Anthropic、AI 署名或共同作者 trailer。

5. 如果发现历史提交中存在上述信息，允许在用户明确授权后通过历史重写删除这些提交信息；重写历史前必须提醒用户该操作会改变 commit hash，并可能需要 force push。
